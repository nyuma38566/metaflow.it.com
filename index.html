Broker Evaluation

Review framework for execution venues

Execution quality

Benchmark fill speed, slippage behavior, order rejection patterns, and spread expansion during macro events or session transitions.

Cost model

Compare raw spread plus commission accounts against all-in spread accounts and document overnight financing on primary symbols.

Symbol coverage

Confirm symbol mapping, contract size, tick value, margin formula, and trading hours for FX majors, metals, indices, or crypto CFDs.

Risk controls

Review stop level restrictions, liquidation thresholds, negative balance protection, and deposit or withdrawal handling windows.

Implementation Notes

Recommended usage pattern

  • Validate every strategy in a historical test environment before attaching it to a live terminal.
  • Adjust spread thresholds, stop distances, and session windows to match the target broker's symbol specifications.
  • Publish installers and client binaries through GitHub Releases so versioning and checksum verification stay explicit.
